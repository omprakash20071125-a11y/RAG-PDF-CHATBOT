# 📄 PDF Chatbot

A RAG-based chatbot that lets you upload a PDF and ask questions about it.
Built with LangChain, FAISS, Groq LLaMA, and Streamlit.

## 🛠️ Tech Stack
- LangChain — RAG pipeline
- FAISS — Vector store for semantic search
- Groq LLaMA 3.1 — LLM for generating responses
- HuggingFace Embeddings — sentence-transformers/all-MiniLM-L6-v2
- Streamlit — UI

## 🚀 How to Run

1. Clone the repo
   git clone https://github.com/your_username/pdf-chatbot.git
   cd pdf-chatbot

2. Install dependencies
   pip install -r requirements.txt

3. Create a .env file and add your Groq API key
   GROQ_API_KEY=your_groq_api_key_here

4. Run the app
   streamlit run streamlit_ui.py
